## Tech Stack

Frontend:
- React.js
- TailwindCSS
- Axios
- React Router DOM
- redux redux-toolkit

Backend:
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose
- JSON Web Tokens (JWT)
- Bcrypt.js

Other:
- Cloudinary (for image upload)
- Dotenv, CORS, Multer

## Installation & Setup

### Backend
```bash
cd server
npm install
npm start

```
### frontend
```bash
cd client
npm install
npm run dev

```
